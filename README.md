# ClientLift - Fitness Client Management System

## Project Description

A lightweight, frontend-only web application for fitness professionals to manage client information and track fitness goals. Features include full CRUD operations with localStorage persistence, real-time client search, and exercise recommendations via the Wger REST API.

## Tech Stack

- **HTML5** - Structure and markup
- **CSS3** - Styling with Flexbox/Grid for responsive design
- **Vanilla JavaScript** - Client-side logic, localStorage, and API integration
- **Wger REST API** - Exercise recommendations
